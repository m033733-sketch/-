# Дача у Мильтюша

Лендинг для аренды двух домиков и бани в деревне Бурмистрово (Новосибирская область, у реки Мильтюш).

## Структура

```
.
├── index.html              # Главная страница (hero, объекты, территория, партнёры, галерея, цены, форма)
├── thanks.html             # Страница «Спасибо за заявку»
├── css/
│   └── style.css           # Все стили (тёмная лесная тема)
├── js/
│   └── main.js             # JS: nav, flatpickr, AOS, sticky-CTA, AI-консьерж, Telegram one-click
├── images/                 # Реальные фото объекта
│   ├── exterior.jpg
│   ├── kitchen-1.jpg
│   ├── kitchen-2.jpg
│   ├── dining.jpg
│   └── bedroom-1.jpg ... bedroom-3.jpg
└── legal/
    ├── privacy-policy.html # Политика обработки персональных данных (152-ФЗ)
    └── oferta.html         # Публичная оферта
```

## Как запустить локально

В папке проекта в терминале:

```bash
python -m http.server 8080
```

Открыть в браузере: `http://127.0.0.1:8080/`

## Что нужно подставить вручную

Поиск по коду «`TODO`» покажет все плейсхолдеры:

- **Телефон** — `+7XXXXXXXXXX` (в `index.html` и `thanks.html`)
- **Telegram личный** — `@your_telegram` → твой ник для брони
- **Telegram канал** — `@your_channel` → канал «Жизнь дачи»
- **TG_USERNAME** в `js/main.js` — тот же ник, что и Telegram личный
- **Партнёры** — имена/услуги/цены/контакты в секции `#partners`
- **`_next` в форме** — после публикации заменить `thanks.html` на полный URL `https://твойдомен/thanks.html`

## Ключевые фичи

- **Hero** с фоном из реального фасада и крупным заголовком в стиле ForestSoul
- **3 карточки объектов** (Домик 1, Домик 2, Баня) с фотографиями и удобствами-чипами
- **Территория** — 4 фичи + интерактивная карта-CTA на Яндекс.Карты
- **Партнёры** (`#partners`) — банщик, повар, рыбак + слот для будущих
- **Галерея** — 7 реальных фото в lightbox-режиме (GLightbox)
- **Тарифы** — 3 карточки с выделением «Популярный»
- **Форма брони** с flatpickr (даты), радио (баня, оплата), калькулятором цены в реальном времени
- **AI-консьерж** — чат-виджет с 13 темами FAQ (rule-based, имитирует AI). Бэкенд под Anthropic API — стаб в `main.js`
- **Sticky-кнопка «Забронировать»** — появляется после hero, скрывается у формы
- **Telegram one-click** — копирует данные брони в буфер, открывает чат
- **Анимации** AOS (fade-up/right/left) с защитой от потери видимости

## Стек

Чистый HTML/CSS/JS, без сборки. Внешние зависимости через CDN:
- [Flatpickr](https://flatpickr.js.org/) — календарь дат
- [GLightbox](https://biati-digital.github.io/glightbox/) — лайтбокс галереи
- Google Fonts: Cormorant Garamond, Playfair Display, Inter

## Деплой

Подойдёт любой статический хостинг:
